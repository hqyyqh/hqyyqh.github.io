【安装记录】neper 多晶体网格生成软件

发表于 2019-05-04 更新于 2019-08-27

概述：安装 neper 前需要安装的 组件 包括

  1. gcc, g++, 2. cmake, 3. gsl, 4. nlopt, 5. povray, 6. gmsh, 7. imagemagick

这里记录了两种环境安装 neper 的过程（基本相同）：1. Win10 下的 Linux 子系统（WSL）2. 桌面发行版 Deepin

WSL 安装 neper

下载 neper，解压，复制到根目录

进入 neper/src/ 目录

mkdir build && cd buid && cmake ..

  • 提示 cmake not found

  • 执行 sudo apt install cmake

再次执行 cmake ..

  • 提示
  • 执行 sudo apt install g++ ，可能还需要 sudo apt install gcc

再次执行 cmake ..

  • 提示

  • 下载安装 GSL，在 GSL 目录下

再次执行 cmake ..

  • 提示

  • 安装 NLOPT，sudo apt-get install libnlopt.dev

再次执行 cmake ..，成功

执行 make -j 多线程加快编译速度

执行 sudo make install，完成安装

运行官方例子gene_gene.sh 提示

例子 sudo sh ./gene_gene.sh

错误提示

  • 安装 povray，执行 sudo apt-get install povray
  • 安装 gmsh sudo apt-get install libgmsh-dev

运行官方例子gene_gene.sh 再提示 convert: not found

  • 安装 imagemagick sudo apt-get install imagemagick
  • 完成

测试：在 build 目录下执行 make test，可以在 build 目录下找到相应的生成文件

Deepin 下安装 neper

在安装neper之前， 需要安装GSL(GNU Scientific Library) ，下载地址：点击打开链接 和NLOPT（不需下载） 。下载Neper。

将GSL和Neper的安装包解压到根目录

更新编译器

安装GSL：cd 到GSL 目录下

安装NLOPT

安装CMAKE

安装neper

cd 到 neper 下的src文件中

安装Gmsh

安装PoV-Ray

测试安装
