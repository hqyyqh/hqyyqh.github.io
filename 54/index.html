前言

  • 一大堆英文 PDF 要概览？
  • PDF 复制出来全是乱七八糟的断行短句？
  • 遇到扫描的英文书籍，只能一个个单词手打翻译？
  • CNKI 翻译、朗文、牛津词典等专业词典查生词和例句？
  • 谷歌翻译、腾讯翻译、有道翻译、百度翻译、搜狗翻译等整个段落？

对论文党来说，以上重(dàn)要(téng)的需求是比较常见的。而这篇教程，就是为了解决这些重(dàn)要(téng)的需求而生。给你30+词典聚合对比翻译、翻译情景模式一键切换、划词&&截图&&复制&&触边翻译

警告：过度依赖翻译工具弊大于利，该啃的硬骨头还是要啃的。

介绍

沙拉查词+Quicker 的组合，就能够完美解决上述大部分的翻译需求。教程中主要介绍 PDF 文档阅读时进行翻译的使用场景

聚合翻译

聚合翻译，喜欢哪个选哪个，如果作为大人的你全都要，那就放在一起对比着看吧。

  • 30+词典
  • 聚合翻译对比

情景模式

情景模式，需要句子或者段落翻译，又需要详细的单词释义？没问题，一键切换。

翻译方式

  • 综合演示
  • 复制翻译模式
  • 触边翻译

想划词翻译？又想截图翻译？还想再偷懒复制就翻译？全都有。

  • 利用 PDF 阅读器内建的复制按钮
    许多 pdf 阅读软件在划词后会出现浮动条，包含复制、高亮等快捷操作，在复制翻译模式下，可通过点击复制来翻译选中的文本

  • 利用 Quicker 的按住左键点右键复制功能
    按住鼠标左键选中文本不松开，接着按下鼠标右键复制


段落排版

PDF 文本的一堆断行怎么搞？嘿嘿，都考虑到了，完美恢复分段，翻译结果不会因为分段问题而产生歧义。

分段后的文本会输出到剪贴板中，Ctrl+v 就可以获取

安装

五星满分的沙拉查词以及作为快捷入口的Quicker都是非常优秀的工具，使用上手简单，并且官网都有详细的中文使用说明，因此本文不再具体介绍二者的具体使用。

安装沙拉查词(Saladict)

大部分的 Chromium 内核浏览器都可以安装沙拉查词扩展，这里以 Chrome 为例。

  1. 假如你可以科学上网，访问 Chrome 商店的沙拉查词 直接安装即可

  2. 浏览器打开 chrome://extensions/shortcuts → 找到沙拉查词-聚合词典划词翻译扩展设置

  3. 在独立窗口中搜索剪贴板内容 设置快捷键 alt+L，右侧选择全局

  4. 测试快捷键能否正常使用，如未生效请彻底重启浏览器。

如果无法科学上网，可以使用谷歌上网助手，Download For chrome 下载安装包，按照安装包内指引完成安装，可以科学上网后，再通过第一步安装。

安装Quicker

  1. 安装 Quicker
  2. Quicker内安装 Quicker 动作文本/截图翻译，执行动作即可。

基本配置结束，以下为可自定义的设置说明

使用

按照安装部分的指引完成后，直接执行动作即可。以下部分为一些具体的自定义设置说明。

选择翻译模式

  • 第一次运行需要选择翻译模式
  • 手动切换

  1. 文本/截图翻译：选中文本时直接弹出沙拉查词窗口进行翻译，未选中时则使用截图翻译。

  2. 复制翻译：复制文本后即自动在沙拉查词窗口中进行翻译。

可以分别安装两次动作，一次使用文本/截图翻译，另一次使用复制翻译，这样可以免去手动切换的繁琐。

右键编辑动作，修改变量【切换模式】的值为 1

选择截图OCR的方式

一般情况下使用默认的截图模式即可，但动作也针对与 PandaOCR 和天若OCR 配合使用做了优化，如需切换可以参考。

Quicker OCR（默认）

动作默认使用 Quicker 自带的截图组件进行截图 OCR。在这个截图模式下，按住 Ctrl 可以多区域连续截图翻译。连续截图可自定义设置为

文本识别接口为百度通用文字识别（高精度版），识别效果非常准确，标点基本能够准确识别，每日使用上限500次；高精度接口次数用完会自动切换到通用文字识别，每日上限五万，基本能够满足需求，部分标点识别不准确。

PandaOCR

PandaOCR 为免费软件，有一些需要了解的已知问题

  • 懒人配置
  • 手动配置

切换方式：右键编辑动作，在右侧变量定义处，修改【OCR模块】变量的值为 2 则选择 PandaOCR，下载 PandaOCR 沙拉查词截图配置.zip，解压即用，无需配置。

手动配置方法

注意：PandaOCR 最低版本要求是 v2.44
设置截图快捷键 Ctrl+M，具体设置按照截图提示，一一修改完成才可以使用

修改文件夹下的 CONFIG.ini 文件

天若OCR

日常使用天若OCR较多的天若专业版用户可以考虑切换到该模式。

  • 第一步 Quicker切换
  • 第二步 天若OCR设置

切换方式：右键编辑动作，在右侧变量定义处，修改【OCR模块】变量的值为 1 则选择天若OCR

开启 识别后复制到粘贴板
快捷键设置：截图识别 F4
快捷键设置：隐藏窗口 ctrl+shift+alt+h


自动分段切换

文本的自动分段依靠标点识别，如果某些文本缺少标点符号，但又不需要分段处理，可以通过修改【自动分段】变量进行切换

快捷键冲突的解决方案

动作提供了方便的快捷键修改方式，右键编辑动作，直接修改右侧相关快捷键变量的值即可。可参考【按键序列】参数

例如 Alt+l 为 “% l” ; Ctrl+l 为 “^l” ; Shift+l 为 “+l” ; Alt+Shift+l 为 “%+l”
