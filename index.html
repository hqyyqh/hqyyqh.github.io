获取知网学位论文 pdf 版本（整理）

发表于 2019-07-12 更新于 2019-08-27

方案一：访问海外版知网，获得原始的 pdf 版本

海外版知网提供 pdf 格式的学位论文下载 http://new.oversea.cnki.net/index/

方案二：安装油猴脚本，调用海外版知网下载链接，获得原始的 pdf 版本

  1. 安装油猴脚本 Tampermonkey，Chrome 浏览器直接在应用商店安装插件。如果无法访问外网，在chrome-extension-downloader.com输入 Tampermonkey 的 ID dhdgffkkebhmkfjojejmpbldmpobfkfo 进行下载离线插件进行安装。
  2. 安装 CNKI PDF 下载脚本，打开 greasyfork.org/zh-CN/scripts/368399-cnki-pdf-download，按照页面提示进行安装即可。
  3. 访问知网的学位论文下载页面，可以看到 PDF 的下载选项。

方案三：caj2pdf gui 版本，获得 caj 转换后的 pdf

caj2pdf 的 GUI 版本。

下载软件直接运行选择 caj 文件即可转换。具体使用看项目主页。

方案四：caj2pdf 命令行版本，获得 caj 转换后的 pdf

安装环境及用法查看 caj2pdf 项目主页。在批量处理 caj 文件时比较方便。
